# 70. 爬楼梯
假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

注意：给定 n 是一个正整数。

示例 1：

输入： 2
输出： 2
解释： 有两种方法可以爬到楼顶。
1.  1 阶 + 1 阶
2.  2 阶
示例 2：

输入： 3
输出： 3
解释： 有三种方法可以爬到楼顶。
1.  1 阶 + 1 阶 + 1 阶
2.  1 阶 + 2 阶
3.  2 阶 + 1 阶

## 
1. 暴力递归
若剩余阶数n>1，爬楼梯方法等于n-1阶和n-2阶时的和。
进行了大量重复计算，复杂度O(2^n)，超时。
2. 动态规划
同1易知，climbstairs(i) = climbstairs(i-1) + climbstair(i-2)；因此只要将i=1 ~ i=n依次计算并储存，即可避免重复计算。
最少只需三位数组存储。
